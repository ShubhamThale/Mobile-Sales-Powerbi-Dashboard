# Mobile Sales Analytics Dashboard (Power BI)

## Project Overview
This project presents an interactive Mobile Sales Analytics Dashboard developed using Microsoft Power BI.

The dashboard helps analyze:
- Total Sales Performance
- Quantity Sold
- Transaction Analysis
- Average Selling Price
- City-wise Sales Distribution
- Brand Performance
- Mobile Model Analysis
- Payment Method Analysis
- Month-to-Date (MTD) Reporting
- Year-over-Year Sales Comparison
---

## Dashboard Features
### KPI Cards
- Total Sales
- Total Quantity
- Total Transactions
- Average Price

### Sales Analysis
- Sales by City
- Sales by Month
- Sales by Day

### Product Analysis
- Sales by Mobile Model
- Brand-wise Performance

### Customer Insights
- Customer Rating Distribution

### Payment Analysis
- UPI
- Credit Card
- Debit Card
- Cash

### Advanced Reporting
- Month-to-Date (MTD) Analysis
- Same Period Last Year Comparison
